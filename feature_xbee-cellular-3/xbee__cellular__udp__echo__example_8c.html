Detailed Description

Sends a UDP datagram to Digi’s public echo server (52.43.121.77:10001) and prints the echoed payload when it comes back.

The flow implements the Extended-Socket “UDP” example: 1) Socket Create (0x40, protocol 0x02) 2) Socket Bind (0x46, choose a local port) 3) Socket SendTo (0x45) --> TX-Status (0x89) 4) Socket ReceiveFrom (0xCE) --> our RX callback 5) Socket Close (0x43) --> Close-Resp (0xCF)
